　 [このデータパックファイルについて]
1.このデータパックの作者は"せんば"です。
2.既に先人が似たようなの作ってますが、"スコアボードなどをワールドに残したくない強い思い"から新たに自分で自分用に作りました


　 [使い方]

 <クラフト方法>
1. 作業台､額縁､金塊を重ねて置くことで作業台が出てきます
2. 出てきた作業台を設置するだけでクラフト出来る環境が整います｡

 <レシピの増やし方>
1. このデータパックを#minecraft:mainに接続します
2. 完成スロットの左隣にコマンドブロックを入れれば生成モードになるのでその状態でレシピをセットします
3. それを樽の上*に設置するだけで専用のコマンドが生成されます｡生成されたコマンドはゲームアウトプットから取り出して下さい｡
4. 出てきたコマンドの下半分(線で区切られてます)を#senba.crafter:recipeに接続したファンクションに入れます
5. 出てきたコマンドの上半分(線で区切られてます)を下半分で実行させるファンクション(ご自分で設定)に入れます

 <レシピの改造>
 ]アイテムの複製:
"/data modify"を使い完成品スロット(Slot:15b)のnbtを複製したいアイテムからコピーする
nbtをコピーする際に一つずつ分けなければならない。(copy_nbtなんて知らない)
 ]アイテムの一部返却
"craft"を実行してしまうとtagが無くなるので実行前に置き換える
実行するのと同じ条件で､idを"modify set"で置き換える。
数は1大きくして置き換える必要がある
 ]アイテムのタグ化
タグ付けしたいアイテムにそれぞれtagを付与し､それを一括で検知する。
タグは樽のアイテムに付与してしまうと後でめんどくさいことが起こるので､アイテムフレームのデータにコピーしてあるアイテムを使う。
"Item.tag.senba-Items"に樽の中がコピーされてある。
 ]一括クラフトがロックされたレシピ(クラフト時に一部アイテムが戻ってくるとかで)
"tag @s[type=!player,tag="#レシピタグ"] add senba.crafter-rook"をタグ､"senba.crafter-recipe"が入る時に入れる
とても説明しづらい...


 <小ネタ>
 ]数を1減らす (正数)
execute store result block ~ ~ ~ Items[{Slot:15b}].Count byte 0.999999 run data get block ~ ~ ~ Items[{Slot:15b}].Count 1
 ]数を1増やす (正数)
execute store result block ~ ~ ~ Items[{Slot:15b}].Count byte -1 run data get block ~ ~ ~ Items[{Slot:15b}].Count -1.00001
